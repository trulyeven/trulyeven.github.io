## 사용법

### 블록, 인라인

```markdown
$$a+b = c$$
```
`$$`,`\[...\]`으로 수식을 감싸면 블록요소가 되고 가운데 정렬이 된다
$$a+b=c$$


```markdown
$y=ax+b$
```
$y=ax+b$ &nbsp;&nbsp;&nbsp;&nbsp;👍

`$`,`\(...\)` 으로 수식을 감싸면 인라인요소로 넣을 수 있다


### 분수

```markdown
$$\frac{a+b+c}{a*c}$$
```
`\frac`{분자}{분모}
$$\frac{a+b+c}{a*c}$$

#### 시그마
```markdown
$\sum_{i=1}^n$
```
$\sum_{i=1}^n$
